# TikTok Shop Sentiment Analyzer
## Somethinc vs Skintific — Data Analyst Portfolio Project

**Dashboard interaktif** analisis sentimen review TikTok Shop untuk 2 brand skincare teratas Indonesia.

### Fitur Analisis
1. 📊 **Sentiment Overview** — Distribusi sentimen & rating per brand
2. 🧴 **Product Analysis** — Tingkat komplain per produk, aspek paling bermasalah
3. 🏆 **Competitive Benchmark** — Radar perbandingan aspek, gap sentimen
4. 📦 **Operational Issues** — Dampak logistik & CS terhadap rating
5. 💰 **Price & Affiliate** — Sensitivitas harga, efek afiliator
6. 🔍 **Anomaly Detection** — Fake review detection, pola temporal
7. ❤️ **Customer Insights** — Loyalty, repeat purchase, R&D signals

### Tech Stack
- **Data:** Python (synthetic data generator)
- **Sentiment:** InSet Lexicon Indonesia
- **Dashboard:** HTML + CSS + Chart.js
- **Deploy:** Vercel
